# Syntecxhub Note App

A simple web app to add and delete notes using HTML, CSS, and JavaScript.

## Demo

Open index.html in your browser to try the app. You can add notes and delete them.

## Features

- Add notes
- Delete notes
- Clean and simple UI

## Technologies

- HTML
- CSS
- JavaScript

## Run locally

1. Clone the repository:
   git clone https://github.com/Anugya345/Syntecxhub_Note_App.git
2. Change into the project directory:
   cd Syntecxhub_Note_App
3. Open index.html in your browser (double-click the file or use a local server like Live Server in VS Code).

## Project structure

- index.html - main HTML file
- css/ - stylesheet(s)
- js/ - JavaScript files (if present)
- assets/ - images or other assets (if present)

Note: adjust paths above if your files are organized differently.
